# What kind of apple? 

## Overview
A mobile app that uses image recognition to identify apple varieties from photos. Users simply take a picture of an apple and receive instant identification with basic information about the variety.

## Problem
People often encounter unfamiliar apple varieties and have no easy way to identify them, leading to uncertainty about taste, uses, and characteristics

## Target User
Apple enthusiasts, grocery shoppers, farmers market visitors, and curious consumers who want to identify apple varieties

## Features
- Camera integration for taking apple photos
- Image recognition engine for apple identification
- Database of common apple varieties
- Basic apple information display (name, characteristics)
- Simple photo gallery of identified apples
